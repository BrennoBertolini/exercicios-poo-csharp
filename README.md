# Exercícios de Programação Orientada a Objetos em C#

Este repositório contém uma coleção de exercícios desenvolvidos com o objetivo de praticar os principais conceitos de Programação Orientada a Objetos (POO) utilizando a linguagem C#.

## Conteúdos abordados

- Classes e Objetos
- Propriedades
- Construtores
- Métodos
- Encapsulamento
- Composição de classes
- Organização de código

## Exercícios implementados

- Livro
- Passagem
- Conta Bancária
- Funcionário
- Retângulo
- Filme
- Produto (controle de estoque)
- Pedido
- Consulta (com reagendamento)
- Produto Digital (com informações técnicas)

Cada exercício possui sua própria classe, todas localizadas no arquivo `Exercicios.cs`.

## Como executar

No arquivo `Program.cs`, existe um método específico para cada exercício.

Para executar um exercício:
1. Abra o arquivo `Program.cs`
2. Descomente apenas o método do exercício desejado no `Main`
3. Execute o projeto

Exemplo:
```csharp
ExecutarLivro();
// ExecutarPassagem();

Objetivo

Este projeto foi desenvolvido exclusivamente para fins educacionais, com foco no aprendizado e consolidação dos fundamentos de POO em C#.
